
1. Number of Missing Values
After upsampling monthly patient vital signs to daily frequency, each variable contained 323 missing values. Since the resulting time series had 365 days and only 12 month-end observations, approximately 88.5% of all values were missing. This high percentage is expected because daily rows are created where no data originally existed.
2. Percentage Missing  
The missingness proportion was consistent across all vital-sign features (temperature, heart rate, blood pressure, weight). Each variable had 323 missing values out of 365 rows, resulting in 88.5% missingness.

3. Imputation Methods Used  
I tested four approaches:  
    - Forward fill (ffill)  
    - Backward fill (bfill)  
    - Linear and time-based interpolation  
    - Rolling-mean imputation  
The primary method ultimately selected was forward fill.

4. Rationale for the Selected Method  
Forward fill was chosen because monthly vital signs typically change gradually rather than abruptly. Forward fill preserves realistic physiological stability across days while avoiding artificial assumptions about how the values should evolve within a month. It is also transparent, easy to interpret, and avoids over-smoothing.

5. Pros and Cons of the Approach  
Pros:
   - Simple and intuitive  
   - Preserves real observed values  
   - Creates stable daily continuity  
   - Avoids artificial trends from interpolation  
Cons:
   - Assumes vitals remain stable through the month  
   - Cannot model true day-to-day variation  
   - May smooth away meaningful short-term changes
